---
title: "Positividad de Campañas"
description: "Retorna el porcentaje de respuestas positivas por campaña de un negocio en un período dado"
api: "GET /estadisticas/obtener/positividadCampaigns/{businessId}"
---

Retorna el índice de positividad por campaña: cuántos mensajes enviados recibieron una respuesta positiva del contacto. Permite identificar qué campañas generan mayor engagement.

## Autenticación

Requiere el encabezado `Authorization: Bearer <token>` con el token del negocio.

## Parámetros de ruta

<ParamField path="businessId" type="number" required>
  ID del negocio cuyas campañas se desean analizar.
</ParamField>

## Parámetros de consulta

<ParamField query="token" type="string" required>
  Token de acceso de Meta Ads del negocio.
</ParamField>

<ParamField query="startDate" type="string" required>
  Fecha de inicio del período en formato ISO 8601 (ej. `2026-04-01T00:00:00Z`).
</ParamField>

<ParamField query="endDate" type="string" required>
  Fecha de fin del período en formato ISO 8601 (ej. `2026-04-30T23:59:59Z`).
</ParamField>

## Respuesta

<ResponseField name="data" type="array">
  Lista de campañas con sus métricas de positividad.

  <Expandable title="Objeto de positividad">
    <ResponseField name="nombre" type="string">Nombre de la campaña.</ResponseField>
    <ResponseField name="mensaje_enviado" type="number">Total de mensajes enviados en la campaña.</ResponseField>
    <ResponseField name="respuesta_positiva" type="number">Cantidad de respuestas positivas recibidas.</ResponseField>
    <ResponseField name="porcentaje_positividad" type="number">Porcentaje de positividad (0–100).</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET "https://beta.api-iobot-desarrollo.com/estadisticas/obtener/positividadCampaigns/1?token=META_TOKEN&startDate=2026-04-01T00:00:00Z&endDate=2026-04-30T23:59:59Z" \
  -H "Authorization: Bearer TU_TOKEN"
```
</RequestExample>
